第九章 多元函数微分法及其应用

对照: 同济大学《高等数学》第七版下册 第九章
一元函数 y=f(x) 升级到多元 z = f(x, y) — 导数也跟着推广.

9.0 整章在干啥?

一元函数 (中学/上册) 你已经会:

  • 给定 y = x², 求导数 dy/dx
  • 用导数找极值 (最高点/最低点)
  • 求切线方程

    • 现在世界更复杂: 山的高度不只跟"东西方向" x 有关, 还跟"南北" y 有关 → z = f(x, y) 是个曲面. 怎么算这种"多变量函数" 的导数+极值+切平面? — 这章答.

    对你的意义: AI 神经网络训练靠梯度下降 — 而梯度 = 多元函数的偏导. 这章直接对应 ML 数学基础.

    9.1 多元函数的基本概念

    知识点 1: 二元函数定义

    一元: 一个输入一个输出. y = f(x) 比如 y = x².

    二元: 两个输入一个输出. z = f(x, y) 比如 z = x² + y² (碗形曲面).

    生活类比:

  • 一元: 你的工资 = f(工龄) — 只看工龄
  • 二元: 工资 = f(工龄, 学历) — 两个都看

    • 定义域: 满足函数有定义的 (x, y) 集合. 比如 z = √(1 - x² - y²) 的定义域是单位圆盘内.

    例题 1: z = (1)/(√(4 - x² - y²)) 的定义域? 分母不能为 0 + 根号下大于 0: 4 - x² - y² > 0, 即 x² + y² < 4 (半径 2 开圆盘).

    ---

    知识点 2: 二元函数的极限

    生活类比: 一元极限是"从 x 左右靠近 x₀, y 趋向某值". 二元呢? 从任何路径 靠近 (x₀, y₀) 都得趋向同一个值, 极限才存在.

    定义 (直观版): 不管 (x, y) 怎么靠近 (x₀, y₀) (直线 / 曲线 / 螺旋), f(x,y) 都趋于 L, 写: lim_((x,y) → (x₀, y₀)) f(x, y) = L

    反例 (路径不同极限不同 → 极限不存在): f(x,y) = (xy)/(x² + y²), 沿 y=0 极限 = 0; 沿 y=x 极限 = 1/2 → 极限不存在.

    ---

    知识点 3: 连续性

    如果 lim_((x,y) → (x₀, y₀)) f(x,y) = f(x₀, y₀), 称 f 在该点连续.

    生活类比: 山地表面没断崖, 就连续. 有断崖, 不连续.

    ---

    9.2 偏导数 (本章核心)

    知识点 4: 偏导数定义

    一元导数 是函数沿 x 方向的变化率. 二元偏导数 是函数沿 x 或 y 单一方向的变化率.

    对 x 的偏导 (把 y 当常数, 对 x 求一元导数): (∂ z)/(∂ x) = lim_(Δ x → 0) (f(x+Δ x, y) - f(x, y))/(Δ x)

    记号: f_x(x, y), ∂ z / ∂ x, z_x 都是它.

    生活类比: 站在山坡某点, 只往东走一小步, 高度变化率 = ∂ z / ∂ x. 只往北走一小步, 高度变化率 = ∂ z / ∂ y.

    求法核心: 对 x 求偏导, 把 y 看成数字, 用一元导数规则; 对 y 求偏导, 把 x 看成数字.

    例题 2: z = x² + 3xy + y², 求 ∂ z / ∂ x 和 ∂ z / ∂ y.

  • ∂ z / ∂ x: 把 y 当数, 对 x 求导: 2x + 3y · 1 + 0 = 2x + 3y.
  • ∂ z / ∂ y: 把 x 当数, 对 y 求导: 0 + 3x + 2y = 3x + 2y.

    • 例题 3: z = e^(xy), 求两偏导.

  • ∂ z / ∂ x: 对 x 求导, y 是数. ∂ e^(xy) / ∂ x = y · e^(xy).
  • ∂ z / ∂ y: x · e^(xy).

    • 例题 3-2: z = x^y (含两个变量的指数), 求两偏导.

  • 对 x 求偏导, y 是常数 → 类似 x 的 y 次方求导: ∂z/∂x = y · x^(y-1)
  • 对 y 求偏导, x 是常数, 用对数微分法: z = x^y, ln z = y ln x, (1/z)·∂z/∂y = ln x, 所以 ∂z/∂y = x^y · ln x.

    • 偏导数的几何意义: ∂z/∂x 在点 (x₀, y₀) 处 = 曲面 z = f(x,y) 被平面 y = y₀ 截出的曲线, 在该点的切线斜率.

    生活类比: 山坡上"切一刀" (用一个平行 xz 的垂直墙切下来), 看墙上画出的"轮廓线" 的斜率 — 就是 ∂z/∂x.

    例题 3-3: 一个气球温度 T(x,y) = 100 - x² - 2y², 在 (1, 1) 处 ∂T/∂x = -2 表示啥? 表示在 (1, 1) 这点, 只沿 x 方向移动单位距离, 温度每变 -2 度. (沿 x 走变冷)

    ---

    知识点 4.3: 偏导数的几何意义 (对齐书 §9.2 进一步说明)

    几何意义: ∂z/∂x 在 (x₀, y₀): 用平面 y = y₀ 截曲面 z = f(x,y), 得到曲线 z = f(x, y₀) (一元函数). 该曲线在 x = x₀ 处的切线斜率, 就是 ∂z/∂x|_(x₀,y₀).

    类似 ∂z/∂y 是用 x = x₀ 平面截出曲线在 y = y₀ 处的切线斜率.

    生活类比: 山的高度 z = f(x,y). 站在 (x₀, y₀) 点:

  • 正东方向 (沿 x 轴) 走小步, 高度变化率 = ∂z/∂x
  • 正北方向 (沿 y 轴) 走小步, 高度变化率 = ∂z/∂y

    • 例题 3-4: f(x,y) = x² + y² (碗形). ∂f/∂x = 2x, ∂f/∂y = 2y. 在 (1, 1) 点: ∂f/∂x|_(1,1) = 2, ∂f/∂y|_(1,1) = 2. 意义: 这点沿 x 走, 高度上升率 2; 沿 y 走, 高度上升率 2. 都是正, 说明这点不是底部.

    知识点 4.5: 偏导数存在 ≠ 函数连续

    ⚠️ 重要: 一元里"可导 ⇒ 连续", 二元里偏导都存在不能推出函数连续.

    反例: f(x,y) = xy/(x²+y²) (当 (x,y)≠(0,0)), f(0,0)=0.

  • 在 (0,0) 偏导: f_x(0,0) = lim (f(Δx,0)-0)/Δx = lim 0/Δx = 0. f_y(0,0) 同样 = 0.
  • 偏导都存在 (都是 0).
  • 但 (0,0) 不连续! 沿 y=x 趋近 (0,0): f = x²/(2x²) = 1/2 ≠ 0.

    • 原因: 偏导只看"沿坐标轴"方向, 函数可能在"其他方向"上不连续.

    结论 (后面会用): 二元函数要"好",我们需要更强条件叫"可微"(全微分存在). 可微 ⇒ 连续 + 偏导都存在.

    ---

    知识点 5: 高阶偏导数

    偏导再求偏导 = 二阶偏导.

    (∂² z)/(∂ x²) = 对 x 偏导后再对 x 偏导. (∂² z)/(∂ x ∂ y) = 先对 y 偏导, 再对 x 偏导.

    重要定理: 若 f_(xy) 和 f_(yx) 都连续, 则相等 (混合偏导跟求导顺序无关).

    例题 4: z = x³ + 3x² y + y³, 求所有二阶偏导.

  • z_x = 3x² + 6xy
  • z_y = 3x² + 3y²
  • z_(xx) = 6x + 6y
  • z_(yy) = 6y
  • z_(xy) = 6x (从 z_x 对 y 求)
  • z_(yx) = 6x (从 z_y 对 x 求) — 跟 z_(xy) 相等 ✓

    • ---

    9.3 全微分

    知识点 6: 全微分定义

    一元的微分: dy = f'(x) dx — 自变量小变化引起函数小变化.

    二元全微分: 自变量 (x,y) 同时小变化, 函数的总变化: dz = (∂ z)/(∂ x) dx + (∂ z)/(∂ y) dy

    生活类比: 你在山坡上走一小步, 这一步既有"往东" 也有"往北" 两个分量 — 总高度变化 = 东向变化率 × 东向走的距离 + 北向变化率 × 北向走的距离.

    例题 5: z = x² y + 3y², 求全微分.

  • z_x = 2xy, z_y = x² + 6y.
  • dz = 2xy · dx + (x² + 6y) · dy.

    • 可微的定义 (严格版): Δz - (∂z/∂x · Δx + ∂z/∂y · Δy) = o(ρ), 其中 ρ = √(Δx² + Δy²) 是"靠近"的距离.

    通俗讲就是: 用线性式 (偏导×Δx + 偏导×Δy) 近似 Δz, 误差比 ρ 小一个量级.

    可微 ⇒ 连续 ⇒ 极限存在. 可微是二元函数"光滑性"的标准.

    充分条件: f_x, f_y 在该点连续 ⇒ f 在该点可微 (反之不一定).

    知识点 6.5: 全微分的近似计算 (实用价值)

    思想: 函数 f(x,y) 在 (x₀,y₀) 邻近, 用线性近似: f(x₀+Δx, y₀+Δy) ≈ f(x₀,y₀) + f_x(x₀,y₀)·Δx + f_y(x₀,y₀)·Δy

    例题 5-2: 估算 (1.02)^(2.99) ≈ ? 取 f(x,y) = x^y, x₀=1, y₀=3, Δx=0.02, Δy=-0.01. f(1,3) = 1. f_x = y x^(y-1) → f_x(1,3) = 3. f_y = x^y · ln x → f_y(1,3) = 1·ln 1 = 0. f(1.02, 2.99) ≈ 1 + 3·0.02 + 0·(-0.01) = 1.06. (精确值约 1.0608, 误差<0.001)

    例题 5-3: 求 √(1.02² + 2.97²) 近似值. f(x,y) = √(x²+y²), 取 x₀=1, y₀=3. f(1,3) = √10 ≈ 3.162. f_x = x/√(x²+y²) → f_x(1,3) = 1/√10. f_y = y/√(x²+y²) → f_y(1,3) = 3/√10. Δx = 0.02, Δy = -0.03. 近似 ≈ √10 + (0.02 - 0.09)/√10 = √10 - 0.07/√10 ≈ 3.162 - 0.022 = 3.140.

    ---

    知识点 6.6: 全微分的"叠加思想" (对齐书 §9.3 进一步)

    核心思想: dz 是 dx, dy 引起的两个独立变化的叠加 — 第一项 ∂z/∂x · dx 是"只 x 变 y 不变" 的影响, 第二项 ∂z/∂y · dy 是"只 y 变 x 不变" 的影响.

    这是线性化: 一个复杂的多变量小变化, 拆成"沿坐标轴各方向的小变化" 之和.

    几何意义: 切平面方程: 曲面 z = f(x,y) 在 (x₀, y₀, z₀) 处的切平面: z - z₀ = f_x(x₀,y₀)(x - x₀) + f_y(x₀,y₀)(y - y₀)

    这就是把曲面在该点用 dz = f_x dx + f_y dy 这条"切平面" 替代.

    例题 5-4: 曲面 z = x² + y² 在点 (1, 2, 5) 处的切平面方程. f_x = 2x, f_y = 2y, 在 (1,2): f_x = 2, f_y = 4. 切平面: z - 5 = 2(x-1) + 4(y-2), 整理: 2x + 4y - z = 5.

    进一步 — 法线方程: 过 (x₀, y₀, z₀) 垂直于切平面的直线, 方向 = (f_x, f_y, -1). (x-x₀)/f_x = (y-y₀)/f_y = (z-z₀)/(-1)

    例题 5-5: 上面曲面在 (1, 2, 5) 处的法线方程. 方向 = (2, 4, -1). 法线: (x-1)/2 = (y-2)/4 = (z-5)/(-1).

    ---

    9.4 多元复合函数求导 (链式法则)

    知识点 7: 链式法则

    情形 1: z = f(u, v), u = u(t), v = v(t). 求 dz/dt. (dz)/(dt) = (∂ z)/(∂ u) (du)/(dt) + (∂ z)/(∂ v) (dv)/(dt)

    生活类比: 你的快乐 = f(收入, 健康), 收入和健康都跟时间相关 → 快乐对时间的变化 = (快乐对收入率 × 收入对时间率) + (快乐对健康率 × 健康对时间率).

    情形 2: z = f(u, v), u = u(x, y), v = v(x, y). 求 ∂ z / ∂ x 和 ∂ z / ∂ y. (∂ z)/(∂ x) = (∂ z)/(∂ u) (∂ u)/(∂ x) + (∂ z)/(∂ v) (∂ v)/(∂ x) (∂ z)/(∂ y) = (∂ z)/(∂ u) (∂ u)/(∂ y) + (∂ z)/(∂ v) (∂ v)/(∂ y)

    例题 6: z = u² + v², u = x + y, v = x - y. 求 ∂ z / ∂ x.

  • ∂ z / ∂ u = 2u, ∂ z / ∂ v = 2v
  • ∂ u / ∂ x = 1, ∂ v / ∂ x = 1
  • ∂ z / ∂ x = 2u · 1 + 2v · 1 = 2(u + v) = 2(2x) = 4x.

    • ---

    9.5 隐函数求导

    知识点 8: 单变量隐函数

    方程 F(x, y) = 0 隐含定义了 y = y(x), 求 dy/dx.

    公式: (dy)/(dx) = -(F_x)/(F_y)

    推理: 两边对 x 求全导数, F_x + F_y · dy/dx = 0, 解出.

    例题 7: x² + y² = 1 (单位圆), 求 dy/dx.

  • F = x² + y² - 1
  • F_x = 2x, F_y = 2y
  • dy/dx = -2x/(2y) = -x/y.

    • 知识点 9: 多变量隐函数

    方程 F(x, y, z) = 0 隐含定义了 z = z(x, y). (∂ z)/(∂ x) = -(F_x)/(F_z), (∂ z)/(∂ y) = -(F_y)/(F_z)

    ---

    9.6 方向导数与梯度 (AI 训练直接相关)

    知识点 10: 方向导数

    偏导是沿 x/y 轴变化率, 方向导数是沿任意方向 l→ 的变化率.

    公式: 沿单位方向 l→ = (cosα, cosβ) 的方向导数: (∂ f)/(∂ l) = f_x cosα + f_y cosβ

    生活类比: 山坡上往任一方向走的高度变化率. 北 = 偏导 y; 东 = 偏导 x; 东北 = 两个加权和.

    ---

    知识点 11: 梯度 (gradient)

    定义: 梯度向量 = (所有偏导组成的向量). ∇ f = (f_x, f_y) (二维) ∇ f = (f_x, f_y, f_z) (三维)

    几何意义: 梯度方向 = 函数值增长最快的方向. 模 = 最快增长率.

    生活类比: 山坡上某点, 梯度指向"最陡上坡方向". 反方向 = 最陡下坡 = 梯度下降.

    AI 神经网络训练: 损失函数 = 多元函数, 梯度下降 = 沿 -∇ f 方向更新参数 → 损失逐步降低.

    方向导数跟梯度关系: (∂ f)/(∂ l) = ∇ f · l→ (梯度点乘方向向量)

    例题 8: f(x, y) = x² + 2y², 求在 (1, 1) 处沿 l→ = (1, 1) 方向的方向导数.

  • f_x = 2x, f_y = 4y, 在 (1,1): ∇ f = (2, 4)
  • 单位化 l→⁰ = (1/√(2), 1/√(2))
  • 方向导数 = ∇ f · l→⁰ = 2/√(2) + 4/√(2) = 6/√(2) = 3√(2).

    • 例题 8-2: f(x,y) = x² + y², 在 (1, 2) 处:

  • 沿哪个方向变化最快? — 沿 ∇f = (2, 4), 单位化 = (1/√5, 2/√5).
  • 最大变化率 = |∇f| = √20 = 2√5.
  • 沿哪个方向不变? — 跟 ∇f 垂直方向, 即 (2, -1)/√5 或 (-2, 1)/√5.

    • 核心结论 (AI 直接用):

  • 梯度方向 = 增长最快方向, |∇f| = 最大变化率
  • 负梯度方向 = 下降最快方向 → 这就是梯度下降算法用的方向
  • 跟梯度垂直的方向 = 函数瞬时不变方向 → "等高线" 切线方向

    • 梯度下降算法核心 (神经网络训练):

  • 当前参数 θ_k, 算损失函数 L(θ_k) 的梯度 ∇L
  • 沿负梯度方向走一小步: θ_(k+1) = θ_k - η · ∇L (η = 学习率)
  • 重复, 直到 ∇L ≈ 0 (到达局部极小)

    • 例题 8-3 (梯度下降模拟): L(x,y) = x²+y², 起点 (4, 3), η = 0.1, 走 2 步. ∇L = (2x, 2y). 在 (4,3): ∇L = (8, 6). 第 1 步: (4,3) - 0.1·(8,6) = (4-0.8, 3-0.6) = (3.2, 2.4). 第 2 步: ∇L 在 (3.2, 2.4) = (6.4, 4.8). 走: (3.2 - 0.64, 2.4 - 0.48) = (2.56, 1.92). 逐步逼近 (0, 0) 这个最小点.

    ---

    9.7 多元微分的几何应用

    知识点 12: 空间曲线的切线

    参数曲线 r→(t) = (x(t), y(t), z(t)), 切向量 = r→'(t) = (x'(t), y'(t), z'(t)).

    切线方程 (点向式): 用切向量做方向向量.

    ---

    知识点 13: 曲面的切平面

    曲面 F(x, y, z) = 0, 法向量 = ∇ F = (F_x, F_y, F_z).

    切平面方程: 用 ∇ F 做法向量, 套点法式. F_x(x₀, y₀, z₀)(x - x₀) + F_y(...)(y - y₀) + F_z(...)(z - z₀) = 0

    例题 9: 球面 x² + y² + z² = 14 在点 (1, 2, 3) 处的切平面.

  • F = x² + y² + z² - 14, ∇ F = (2x, 2y, 2z), 在 (1,2,3) = (2, 4, 6).
  • 切平面: 2(x-1) + 4(y-2) + 6(z-3) = 0, 即 x + 2y + 3z = 14.

    • ---

    9.8 多元函数的极值

    知识点 14: 极值的必要条件

    可微函数 z = f(x, y) 在 (x₀, y₀) 取得极值, 则: f_x(x₀, y₀) = 0 且 f_y(x₀, y₀) = 0

    (梯度为零)

    驻点: 满足 f_x = f_y = 0 的点. 极值点必是驻点, 但驻点不一定极值 (可能是鞍点).

    ---

    知识点 15: 极值的充分条件 (二阶判别)

    驻点 (x₀, y₀) 处求:

  • A = f_(xx), B = f_(xy), C = f_(yy)
  • 判别式 D = AC - B²

    • 判别:

  • D > 0 且 A > 0: 极小值
  • D > 0 且 A < 0: 极大值
  • D < 0: 鞍点 (非极值)
  • D = 0: 此法失效, 另想办法

    • 例题 10: f(x,y) = x² + y² + 2x - 4y + 3, 求极值.

  • f_x = 2x + 2, f_y = 2y - 4. 解 f_x = f_y = 0 → 驻点 (-1, 2).
  • A = f_(xx) = 2, B = f_(xy) = 0, C = f_(yy) = 2.
  • D = 4 - 0 = 4 > 0, A > 0 → 极小值.
  • 极小值 = f(-1, 2) = 1 + 4 - 2 - 8 + 3 = -2.

    • ---

    知识点 16: 条件极值 — 拉格朗日乘数法

    求 f(x, y) 在约束 φ(x, y) = 0 下的极值.

    构造拉格朗日函数: L = f + λ φ, 然后求驻点: L_x = 0, L_y = 0, L_λ = 0

    生活类比: 给定预算 (约束), 求快乐最大 (目标函数). 拉格朗日乘数 λ 就是"边际快乐" = 多 1 元预算能多多少快乐.

    例题 11: 求 f(x, y) = xy 在 x + y = 1 约束下的极值.

  • L = xy + λ(x + y - 1)
  • L_x = y + λ = 0 → y = -λ
  • L_y = x + λ = 0 → x = -λ
  • L_λ = x + y - 1 = 0 → -2λ = 1 → λ = -1/2, x = y = 1/2
  • 极大值 = 1/4.

    • 例题 11-2 (经典几何题): 在椭圆 x²/4 + y² = 1 上求一点, 距原点最远 + 最近. 目标函数 = 距离平方 f(x,y) = x² + y² (避免根号). 约束: g(x,y) = x²/4 + y² - 1 = 0. L = x² + y² + λ(x²/4 + y² - 1)

  • ∂L/∂x = 2x + λx/2 = x(2 + λ/2) = 0 → x = 0 或 λ = -4
  • ∂L/∂y = 2y + 2λy = 2y(1+λ) = 0 → y = 0 或 λ = -1
  • 约束: 椭圆方程

    • 讨论:

  • x = 0, y² = 1 → y = ±1, 距离 = 1.
  • y = 0, x² = 4 → x = ±2, 距离 = 2.

    • 最近 = 1, 最远 = 2 (椭圆短半轴 1, 长半轴 2).

    例题 11-3 (实用问题): 用 12 平方米材料做长方体盒子无盖, 求最大体积. 设长 x, 宽 y, 高 z, 体积 V = xyz. 约束: xy + 2xz + 2yz = 12 (无盖, 底 + 4 个侧面). 拉格朗日: L = xyz + λ(xy + 2xz + 2yz - 12) 偏导联立: yz + λ(y + 2z) = 0 xz + λ(x + 2z) = 0 xy + λ(2x + 2y) = 0 解得 x = y = 2, z = 1, V = 4 立方米.

    知识点 17: 多元函数最值 (在有界闭区域上)

    算法 (求闭区域 D 上 f 的最值):

  • 内部驻点: 求所有 f_x = f_y = 0 的点
  • 边界极值: 在 D 的边界上, 化为一元函数 + 求一元极值
  • 端点
  • 全部代入 f, 比大小

    • 例题 11-4: 求 f(x,y) = xy 在 D = {x² + y² ≤ 1} 上的最值. 内部驻点: f_x = y = 0, f_y = x = 0 → (0,0), f = 0. 边界 x² + y² = 1, 用参数 x = cosθ, y = sinθ: f = cosθ sinθ = sin(2θ)/2. 最大值 = 1/2 (在 θ=π/4, 即 (√2/2, √2/2)), 最小值 = -1/2. 结论: max = 1/2, min = -1/2.

    9.9 二元函数的泰勒公式 (对齐书 §9.9)

    知识点 17: 二元泰勒公式

    一元有 f(x) ≈ f(a) + f'(a)(x-a) + f''(a)(x-a)²/2 + ... 这种"用多项式逼近函数"的展开. 二元同样有:

    f(x, y) ≈ f(a, b) + [f_x(a,b)(x-a) + f_y(a,b)(y-b)] + (1/2!)[f_xx(x-a)² + 2 f_xy(x-a)(y-b) + f_yy(y-b)²] + ...

    生活类比: 像爬山, 站在 (a,b) 看附近地形, 一阶展开 = 平面 (切平面), 二阶展开 = 抛物面 (凹凸感), 高阶继续精细化.

    知识点 18: 二阶判别式的"由来" (对齐 §9.9 二)

    之前 (知识点 15) 用 D = AC - B² 判极值类型. 这个就是二阶泰勒展开: A=f_xx, B=f_xy, C=f_yy, 是 Hessian 矩阵的项. D > 0 + A > 0 → Hessian 正定 → 极小值.

    应用: AI/ML 优化里用 Hessian 判断"局部最小 vs 鞍点", 是同一套数学.

    9.10 最小二乘法 (对齐书 §9.10)

    知识点 19: 最小二乘法

    问题: 一堆数据点 (xᵢ, yᵢ), 怎么找一条直线 y = ax + b 最贴合?

    思想: "误差平方和最小". 残差 = (yᵢ - axᵢ - b)², 加起来求最小: S(a, b) = Σᵢ (yᵢ - axᵢ - b)²

    对 a, b 求偏导 = 0, 解出: a = (n·Σxy - Σx·Σy) / (n·Σx² - (Σx)²) b = (Σy - a·Σx) / n

    为啥用平方和: ① 处理"正负误差不抵消" ② 求导方便 ③ 跟正态分布最大似然估计一致.

    例题 12: 数据点 (1,2), (2,3), (3,5), (4,4), 拟合 y = ax + b. n=4, Σx=10, Σy=14, Σxy=39, Σx²=30. a = (4·39 - 10·14)/(4·30 - 100) = (156-140)/(120-100) = 16/20 = 0.8. b = (14 - 0.8·10)/4 = 6/4 = 1.5. 拟合直线: y = 0.8x + 1.5.

    AI 直接用: 线性回归就是最小二乘. 神经网络反向传播找梯度也是这个思想推广.
